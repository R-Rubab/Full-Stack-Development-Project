# 🌐 Responsive Landing Page

A modern and fully responsive landing page developed using **HTML5**, **CSS3**, and **JavaScript**. This project was created as part of the **DecodeLabs Full Stack Development Internship – Week 1** to demonstrate responsive web design and frontend development skills.

---

# 📖 Project Overview

The Responsive Landing Page is designed with a clean and modern user interface that adapts to different screen sizes. It provides a smooth user experience across desktops, tablets, and mobile devices.

---

# ✨ Features

* ✅ Fully Responsive Design
* ✅ Modern Navigation Bar
* ✅ Hero Section
* ✅ About Section
* ✅ Services Section
* ✅ Contact Section
* ✅ Smooth Scrolling
* ✅ Mobile-Friendly Layout
* ✅ Clean UI Design
* ✅ Interactive Buttons
